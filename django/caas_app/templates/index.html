API

Steakscorp CaaS is inspired by Tom Dionysus' FOAAS project, and provides a RESTful solution to the problem of finding cat-facts to send to your friends.

Results are returned as a JSON string. Requests are made using the HTTP GET method, and parameters can be chained together as normal in the request body.

Parameter Input Example output Use
No parameters N/A

exclude comma-seperated source names (default include all sources)
